## 1.4 Resumen de los formatos y conversión de archivos

| Etapa del flujo                                  | Formato                              | Descripción                                                  | Conversión / Herramienta típica                              |
| ------------------------------------------------ | ------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **1. Entrada cruda (lecturas de secuenciación)** | **AB1**                              | Lecturas de secuenciación Sanger (electroferogramas)         | AB1 → FASTA o FASTQ (con **Biopython**, **SeqTrace**, **Chromas**) |
| **2. Secuencias **                               | **FASTA**                            | Secuencias de ADN o proteínas sin calidades                  | VCF → FASTA                                                  |
|                                                  | **FASTQ**                            | Lecturas con calidades (NGS)                                 | FASTQ → SAM/BAM (alineamiento con **BWA**, **Bowtie2**)      |
| **3. Alineamientos**                             | **SAM/BAM**                          | Alineaciones contra genoma de referencia                     | BAM → VCF (llamada de variantes con **GATK**, **bcftools**)  |
| **4. Variantes**                                 | **VCF**                              | Variantes genéticas identificadas (SNPs, INDELs)             | VCF → PLINK (**plink --vcf**), o → BED (regiones de variantes)<br />VCF → (**vcfR, adegenet**) Geneind/Genelight |
| **5. Regiones genómicas**                        | **BED**                              | Coordenadas genómicas (anotaciones o variantes)              | BED ↔ GFF, BigBed                                            |
| **6. Análisis poblacional**                      | **PLINK (.ped/.map/.bed/.bim/.fam)** | Formato para análisis de genotipos                           | PLINK ↔ VCF, Genepop, Geneind                                |
|                                                  | **Genepop**                          | Datos genéticos poblacionales (frecuencias, loci)            | PLINK → Genepop (con **PGDSpider**)                          |
|                                                  | **Geneind / Genelight**              | Formatos específicos para genética de poblaciones (genotipos individuales) | PLINK/Genepop → Geneind/Genelight (conversión mediante **R packages adegenet/poppr**) |
